# Mini-game-number
Міні гра "вгадай число", консольна гра

# Як встановити гру?

Завантажуєм архів, розархівовуємо файли в окрему папку.
Завантажуєм Visual Studio 2022 та відкриваєм проект, по шляху D:\Mini-games\mini-game
знаходим файл main.cpp
В верхній панелі Visual Studio запускаєм проект, гра працює
